**Prescription Management System (React + Spring Boot Web Application):**
<br>A web application that enables doctors to send prescriptions to pharmacies electronically. Doctors can log in, select a pharmacy, and submit patient prescriptions. Pharmacies receive prescriptions in real-time, update their status, and automatically send email notifications to patients about their prescription progress.
